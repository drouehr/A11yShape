Welcome to Code2Fab!
Quick Navigation and Operation Guide using Keyboard Shortcuts:
- Model Generation: <Shift+Alt+G> to render the model and generate a description.
- View Switching: <Shift+Alt+V> to view the model from a different angle.
- List Model Parts: <Shift+Alt+C> to update the list of model parts.
- Model Part Selection: Hold <Shift+Alt> and use <Up/Down Arrow Keys> to select model parts.
- Select Full Model: <Shift+Alt+M> to select the full model instead of model parts.
- Save Model: <Shift+Alt+S> to save as a .scad file.
- Open File: <Shift+Alt+O> to load a .scad file.
- Toggle Column: Hold <Shift+Alt> and use the keys <1, 2, 3> to move to a different column.
- Jump to Editor: <Shift+Alt+E> to go to the code editor.
- Jump to Model Description: <Shift+Alt+D> to go to the generated description.
- Help: <Shift+Alt+H> to view this instructions section.

Code2Fab aims to enhance your efficiency from code to creation with powerful, intuitive tools tailored for designers, engineers, and enthusiasts alike.
